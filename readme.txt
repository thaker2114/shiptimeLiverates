== Description ==

This plugin allows you to use real-time shipping rates provided by third-party shipping services.
You can create multiple shipping services per store.
Calculations can be disabled for any service.
You can also specify custom origin addresses for each shipping service.

== Installation ==

1. Upload the `live-shipping-4-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin require the Woocommerce plugin? =

Yes, the Woocommerce plugin must be installed and active.
